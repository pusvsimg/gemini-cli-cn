# 图标文件说明

本项目在 `manifest.json` 中引用了以下图标文件：

- `icon-192x192.png` (192x192 像素)
- `icon-512x512.png` (512x512 像素)

这些图标用于 PWA (渐进式 Web 应用) 安装功能。如果您要部署此网站并希望支持 PWA 功能，请创建相应尺寸的 PNG 图标文件并放置在项目根目录中。

推荐使用以下命名和尺寸：
- `icon-192x192.png`
- `icon-512x512.png`

图标设计建议：
- 使用透明背景
- 图标主体内容占据图像 80% 的空间
- 保持简洁的设计以适应不同尺寸
